### Makefile

| Command            | Detail                |
|--------------------|-----------------------|
| `make all`         | compile all binaries  |
| `make doc`         | compile documentation |
| `make run`         | run code              |
| `make run-<model>` | run code with model   |

One may modify python scripts in /main/script to change parameters.
